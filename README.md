# 作业报告：全球人均温室气体排放趋势可视化

## 1. 可视化方案旨在解答什么问题？

本项目旨在探索 **2014年至2023年间**，全球主要国家（OECD成员国）的**人均温室气体排放量**的变化趋势。

通过该可视化图表，用户可以快速回答以下问题：
*   **对比分析**：在选取的年份中，哪些国家的人均碳排放量较高（如澳大利亚、美国）？哪些国家较低（如比利时、瑞士）？
*   **趋势观察**：近十年来，各国的排放量是呈上升趋势还是下降趋势？（例如：大部分发达国家呈现缓慢下降趋势）。
*   **特定关注**：某一个特定国家（如美国或德国）的排放趋势是怎样的？

## 2. 设计决策依据（可视化编码）

为了清晰地展示数据，我采用了以下视觉编码方式：

*   **X 轴 (位置)**
    *   **编码含义**：时间 (年份)
    *   **设计理由**：使用线性刻度展示时间序列，便于观察排放量的长期变化趋势。

*   **Y 轴 (位置)**
    *   **编码含义**：排放量 (千克 CO2当量/人)
    *   **设计理由**：使用数值刻度，直观展示排放量的高低差异，让用户一眼看出谁多谁少。

*   **线条 (颜色/图例)**
    *   **编码含义**：不同的国家
    *   **设计理由**：使用不同颜色区分国家，配合交互式图例，避免线条过多造成的视觉混乱。

*   **圆点 (标记)**
    *   **编码含义**：具体数据点
    *   **设计理由**：在折线上添加圆点，引导用户视线，方便鼠标悬停查看具体数值。

## 3. 交互功能设计

为了提升用户体验，图表设计了以下交互功能：

*   **🔍 鼠标悬停 (Tooltip)**
    *   当鼠标移动到数据点上时，显示黑色提示框，精确展示该年份的具体排放数值。这解决了折线图难以精确读数的问题。

*   **🚩 图例点击交互 (Highlight/Dim)**
    *   点击底部的国家名称，可以高亮（Highlight）该国家的线条，同时淡化（Dim）其他线条。这解决了“多条线重叠导致看不清某一条线”的问题，让用户能专注分析特定国家。
